# Scientific Reasoning for Deep Tech | PhD Theoretical Physics | Strategy & Quantitative Analytics
Hi there, I am Kay and you stumbled upon my contributions to open source software. I mostly work in `C` and am always eager to optimise the code for speed and stability, using `openMP`, `MPI`, and the force of different compilers and their compile flags. Did you know that you often can easily reduce the runtime of scientific computations by 5%~25% just by using [Profile-Guided Optimisations](https://www.linkedin.com/posts/kb-lehnert_i-made-my-scientific-code-123-faster-with-ugcPost-7442312826372689920-rimF/?utm_source=share&utm_medium=member_desktop&rcm=ACoAACainH4BeLFIcHSL9g3jNhlQn6qowHQLIGU)? This takes you 15 minutes, but can save you thousands of hours on supercomputers.
